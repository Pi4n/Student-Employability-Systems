# CCS3402 Assignment (Frontend-only)

This folder contains a **frontend-only** prototype (HTML/CSS/JS) implementing:

- Student Dashboard (radar chart + progress bars)
- Activity Logging (staff + student modules)
- Employability Scoring (Marketability Index)
- Admin Management (Learning Outcomes CRUD)

## How to run

Open `index.html` in your browser.

## Notes

- Data is stored in the browser using `localStorage` (no MySQL required).
- Click **Seed demo data** on any page to populate sample entities from the ERD.
