Holistic Student Marketability (Frontend-only)

This prototype implements the required features using HTML, CSS, and JavaScript. Data is stored in localStorage (no MySQL).

Open Student Dashboard Open Activity Logging Open Admin Management
Student Dashboard
Radar chart + progress bars comparing Academic vs Co-curricular.
Activity Logging
Staff enter courses/activities; students register enrollment and participation.
Employability Scoring
Computes a Marketability Index from holistic data + breadth bonus.
Admin Management
CRUD for university-mandated Learning Outcomes.
Tip: first run seeds demo data using the button below.

Quick start

  1. Click Seed demo data (once).
  2. Go to Activity Logging and register student enrollment and co-curricular participation.
  3. Open Student Dashboard to see the chart + Marketability Index update.
  4. Use Admin Management to edit Learning Outcomes.
