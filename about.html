About this project

The Holistic Student Marketability system is a production-ready full-stack web application designed for tracking and visualizing student employability based on academic and co-curricular achievements.

Architecture

  • Static HTML frontend with Bootstrap 5 dark theme
  • Netlify Serverless Functions (Node.js)
  • Oracle SQL Database (10-table relational model)
  • Server-side data translation layer for backward compatibility

Core data entities

Programs Students Courses Enrollments Learning Outcomes Employability Skills Co-curriculum Skill Mappings
